# Athanor

A self-feeding furnace. Unified local AI system, media server, home automation, game development platform, and creative workshop — built with craft, designed to grow.

## Quick Start with Claude Code

```powershell
cd athanor
claude
/project:orient
```

## Slash Commands

| Command | Purpose |
|---------|---------|
| `/project:orient` | Session startup — read state, find where we left off |
| `/project:research <topic>` | Deep research with citations |
| `/project:decide <decision>` | Create Architecture Decision Record |
| `/project:audit <target>` | Hardware discovery on a node |
| `/project:project <name>` | Switch to a specific project context |

## Structure

```
CLAUDE.md          — Claude Code instructions
docs/VISION.md     — What Athanor is and why (source of truth)
docs/research/     — Research notes
docs/decisions/    — Architecture Decision Records
docs/hardware/     — Audit results and inventory
docs/projects/     — Per-project documentation
projects/          — Project workspaces
```
